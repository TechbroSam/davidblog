## Getting Started

First, install dependencies

# Add your .env variables
DATABASE_URL=

# Admin Credentials
ADMIN_EMAIL=
ADMIN_PASSWORD=

# NextAuth Secret (you can generate one with `openssl rand -base64 32`)
NEXTAUTH_SECRET=

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open at [http://localhost:3000]
